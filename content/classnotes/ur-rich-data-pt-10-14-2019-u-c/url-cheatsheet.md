# URL CheatSheet

## Format
{protocol}://{server.name:port}/the/path/to/file.html?{key=value}&{key2=value2}&{key3=value3}

## Protocol
Protocol: This is the format of the request. In most cases it will be `http` or `https`.
http: HyperText Transfer Protocol
https: Secure Hypert
