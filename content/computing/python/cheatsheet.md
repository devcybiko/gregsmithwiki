# Python CheatSheet

* Variables
	* Use upper and lower case letters
	* Plus "_" and "$" if necessary
	* Assign values with '=' sign
	* can take any kind of value (string, number, etc)
`foo = 1`
`bar = "hiya"`

	* numeric constants (digits, period, more digits)

* Print to the console
`print("string...")`
