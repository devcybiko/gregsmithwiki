# React Setup

1. Install create-react-app
	- `npm install -g create-react-app`
2. Go to the `git` project directory
	3. `cd ~/git`
3. Create a React project
	4. `create-react-app project1`
4. Go to the new project folder
	5. `cd project1`
5. Run the template project
	6. `npm start`
